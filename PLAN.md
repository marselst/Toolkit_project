# Plan

## Version 1

### 1.1 Document Management
- Create an application where users can add documents with different types of extensions:
  - `.txt`
  - `.docx`
  - `.pdf`
  - and others

### 1.2 Search Availability Marking
- Add the feature allowing users to mark documents as **available for search**

### 1.3 Search Mode
- Implement a search mode with a search input field
- The application will find the provided string in files marked as available for search

---

## Version 2

### 2.1 Folder Support
- Modify the main feature by adding folder functionality
- Users can create folders to organize documents
- Users can mark entire folders — all files within will become available for search

### 2.2 File Format Conversion
- Add a second feature: **file format conversion**
- Allow users to convert files between different extensions
